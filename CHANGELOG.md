# Changelog

## v0.1.0 - Public Knowledge Base Foundation

Released 2026-05-30.

### Added

- Public README positioning the project as an open router recovery knowledge base.
- Documentation license using CC BY 4.0.
- Code, schema, and tooling license using MIT.
- Contribution, security, and conduct policies.
- GitHub issue templates for recovery reports and documentation corrections.
- Pull request checklist for evidence boundaries.
- TFTP Recovery Guide.
- OpenWrt Recovery Guide.
- Firmware Selection Guide.
- Common Recovery Failures guide.
- ASUS Recovery Guide.
- TP-Link Recovery Guide.
- NETGEAR Recovery Guide.
- OpenAI Codex for Open Source application notes.

### Maintainer Notes

- Router Recovery for macOS remains a separate commercial app.
- The app is mentioned only as an optional related tool.
- The public repository should remain useful without the app.
- Recovery claims must separate upload completion from completed recovery.
- Device-specific observations must not be generalized without evidence.
