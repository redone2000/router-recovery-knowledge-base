## Application Positioning

Router Recovery Knowledge Base is the open-source project.

Router Recovery for macOS is a separate commercial app. The app can demonstrate long-term maintenance context and real user problems, but it is not the open-source repository being submitted.

## Why This Repository Qualifies

```text
Router Recovery Knowledge Base is an open, evidence-based documentation and tooling project for router firmware recovery, TFTP recovery, OpenWrt/ImmortalWrt recovery, and device-specific troubleshooting. It helps users recover existing hardware safely, avoid firmware-selection mistakes, and reduce unnecessary e-waste. I maintain the guides, schemas, evidence review workflow, and issue triage.
```

## Additional Context

Router Recovery is a commercial macOS app, but this repository is a separate public open-source knowledge base. ## Commercial App Link Boundary

Good wording:

```text
Router Recovery for macOS is an optional guided tool for TFTP-oriented recovery workflows and local recovery attempt records.
```

Avoid:

```text
Download our app to fix your router.
```

The public repository should solve real documentation problems first. App links should be secondary, relevant, and clearly optional.
