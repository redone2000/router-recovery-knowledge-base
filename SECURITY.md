# Security Policy

This project documents recovery behavior for routers and network devices. Please avoid publishing sensitive device, network, or account information.

## Please Do Not Publish

- admin passwords
- cloud account credentials
- serial numbers
- MAC addresses
- private IP screenshots that reveal unrelated devices
- proprietary firmware contents
- exploit instructions unrelated to legitimate device recovery

## Reporting Sensitive Issues

If you find a problem that could expose private data or enable misuse, open a minimal GitHub issue without sensitive details and ask for a private contact path.

## Scope

This repository is a documentation and tooling project. The commercial Router Recovery macOS app is maintained separately and is not covered by this repository's source tree.
