# Code of Conduct

This project follows a practical, evidence-focused collaboration standard.

Be respectful, keep discussions technical, and separate observed facts from assumptions. Router recovery can be stressful because users may be dealing with unusable hardware; help reduce confusion rather than adding certainty where evidence is missing.

Unacceptable behavior includes harassment, personal attacks, spam, credential requests, or instructions that are primarily about unauthorized access rather than legitimate recovery of owned devices.
